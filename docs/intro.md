TestBench Extension for Visual Studio Code connects TestBench and Robot Framework directly inside VS Code.

## What you can do

- Browse TestBench projects, test object versions (TOVs), test cycles, subdivisions, and keywords.
- Generate Robot Framework test suites from TestBench test structures.
- Bidirectionally synchronize Robot Framework keywords between VS Code and TestBench.
- Import Robot Framework test execution results back into TestBench.

## Core requirements

- Visual Studio Code 1.101.0 or newer.
- Python 3.10 or newer.
- TestBench 4.0 or newer.
- Open VS Code workspace folder (required for write operations).

The extension auto-installs these dependencies:

- Python extension (`ms-python.python`)
- RobotCode extension (`d-biehl.robotcode`)
